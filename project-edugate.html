Edugate

Logic Circuit Education Game
Winner of the "Best Educational Hack" at DragonHacks 2023
Developed in Python using Pygames

GitHub

DevPost

Edugate was a hackathon project that I worked on with the help of my two friends for the DragonHacks 2023 hackathon. We decided to develop an educational game that would help teach the importance of circuits, and how every application relies on this basis.

Im a big fan of understanding why things work they way they do, and I think its really cool that gates like NAND and NOR are universal gates. Basically what this means is that every logic gate in existance can be created with just these two gates. And we can of course continue to build more gates to build more complex machines like computers.

The goal of this project was to get young engineers excited and teach them the important of gates. As well as teach them how we can build more important gates. In our game, you are able to build up to a full adder. This hackathon was a really fun experience with me and my friends, and you can see the documentation and game below!
