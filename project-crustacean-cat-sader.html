Itch.io

This game is a 2D rougelike game I created with a friend of mine who is new to programming. This was made for the 2024 Pixel Jam. The theme of this Jam was "Aqua", and what better comes to mind then a cat in the ocean??? My idea for the game was to have a cat in the ocean exploding various enemies with items you collect as you level up. Though with the time limitation I was only able to add one enemy, so I pivoted to a game where you try to kill as many crabs as possible!!

In the game, there are 5 items which I added to help you survive as more and more enemies spawn. As the game plays, the enemies get faster and more health, so eventually it becomes extremely hard to dodge. This was a fun experience for me, and I was able to learn a lot about good game dev design patterns. I also did all of the art for the game!
