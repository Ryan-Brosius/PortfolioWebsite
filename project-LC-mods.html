Mods for Lethal Company

GitHub

Thunderstore

When Lethal Company came out on steam I was a really big fan. I decided that I wanted to import some funny characters into the game for my friends and I. With the help of the LC model replacement API I was able to mod some character replacements into the game. I was also at one point helping the main dev of the API find bugs and fix them on discord.

Below are a few screenshots of me setting up a character to import into lethal company. To import a character (or multiple) you need to build them into an asset bundle that can be injected into the game. I also needed to fix and model bugs, rig them, add their textures, and add jiggle bones for fun. Here is also some of the code I wrote which allows us to inject the models into the game.
